# 💬 React Comments App

This is a simple React project where users can add comments using a form.  
The comments are displayed below the form in a clean Bootstrap UI.

---

## 🚀 Features

- Add new comments
- Form validation using Formik
- Clean Bootstrap UI
- Real-time comment display
- Responsive design

---

## 🛠️ Technologies Used

- React JS
- Bootstrap 5
- Formik
